HTML5 - Basis - Syntax & Vocabulary

<html> <head> <title>Browser Title</title> </head> <body> <h1>Main Title</h1> <img src="images/lille.jpg" /> </body> </html>
  • head, body, title, h1, img are HTML elements
  • html The beginning/container of the whole document
  • head Give informations concerning the document: title, metadata, js scripts, stylesheets...
  • body The beginning of the page content. Define the visible part of the document.
  • An HTML element can be configured with attributes (src)
