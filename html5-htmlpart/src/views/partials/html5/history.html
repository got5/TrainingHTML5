HTML5 - History

  • Created by Tim Berners-Lee in 1980 for the CERN (European Organization for Nuclear Research)
  • Based on SGML (Standard Generalized Markup Language), HTML (HyperText Markup Language) starts with 18 elements.
  • Standards:
    • W3C (World Wide Web Consortium): 1994
    • WHATWG (Web Hypertext Application Technology Working Group): 2004
    • Initially worked together but are now independant, who is right?
