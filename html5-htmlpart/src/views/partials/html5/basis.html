HTML5 - Basis - Doctype

  • The DOCTYPE element allows to specify a DTD (Document Type Declaration).
    • Exemple: 
      <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
  • Previously, 6 DTDs availables to declare your document:
    • 3 versions for each type: transitional, strict, frameset
  • Much more easier with HTML5: <!DOCTYPE HTML>
