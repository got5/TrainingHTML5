HTML5 - More Semantic - Section & Article

  • section:
    • the most generic of the sectioning element
    • defines sections in a document, such as chapters, headers, footers, or any other sections of the document
    • The theme of each section should be identified, typically by including a heading (h1-h6 element)
    • Is usefull to outline the document, use HTML 5 outliner tool
  • article:
    • specifies independent, self-contained content
    • make sense on its own and it should be possible to distribute it independently from the rest of the site
<section> <h2>level of heading = section nesting level</h2> rest of the content </section> <article> <h1>My Information</h1> <p>This object contains data which could be shared through RSS for example</p> </article>
