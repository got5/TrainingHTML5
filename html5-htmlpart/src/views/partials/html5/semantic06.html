HTML5 - More Semantic - Header & Footer

    • header:
      • specifies a header for a document or section
      • should be used as a container for introductory content
      • So it's possible to have several header tags in one document
    • footer:
      • specifies a footer for a document or section
      • should contain information about its containing element, typically the author, copyright ...
  • <article class="small"> <header> <h1>My article title</h1> <p><time datetime="2014-09-01"></time></p> </header> <p>My article is so fine, just read it!</p> <footer> <p>author: john doe</p> </footer> </article> article{ border: 1px solid black; padding: 0.5em; } article h1{ font-size: smaller; text-decoration: underline; }
