• Utiliser le tag script, et utiliser l'attribut src pour préciser son url.
  • Se rappeler que javascript n'est pas un langage compilé, et l'ordre dans lequel sont importés les scripts est donc important
<html> <head> <script src="js/script1.js" type="text/javascript" ></script> <script src="js/script2.js" type="text/javascript" ></script> </head> </html>
