• Specify the url of your file by using the script tag with the src attribute
  • Remember that javascript isn't a compiled language and that the order in which the files are imported is also the order in which they are executed
<html> <head> </head> <body> <script src="js/script1.js" type="text/javascript" ></script> <script src="js/script2.js" type="text/javascript" ></script> </body> </html>
