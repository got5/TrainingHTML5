• There are two event models:
    • Bubble -> From the deeper element to the body
    • Capture -> Opposed to Bubble
  • Bubbling is the only one that currently works in all browsers and jQuery makes it consistant by replacing Capture events by Bubble ones.
  • When an event occurs, an event object is created and passed to listeners
    • Many DOM event subclasses : MouseEvent, FocusEvent, KeyboardEvent
    • Properties : bubbles, cancelable, currentTarget, defaultPrevented, eventPhase, target, timestamp and type
    • Methods : initEvent, preventDefault, stopImmediatePropagation, stopPropagation
