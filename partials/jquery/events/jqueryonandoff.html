• New in jQuery 1.7
  • Replacement for the 3 separate events API : bind, live and delegate
  • bind(), live() and delegate() will remain in jQuery for backwards compatibility
  • Internally bind(), live() and delegate() now use the on() API
  • Chainable with traversal/filter methods (unlike live)
  • No confusion with ES5 Function.bind
  • Possibility to bind to as many events as you want
