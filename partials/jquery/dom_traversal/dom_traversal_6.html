• Official API:
    • prev(): Get the immediately preceding sibling of each element in the set of matched elements, optionally filtered by a selector
    • prevAll(): Get all preceding siblings of each element in the set of matched elements, optionally filtered by a selector.
    • next(): Get the immediately preceding sibling of each element in the set of matched elements, optionally filtered by a selector
    • nextAll(): Get all preceding siblings of each element in the set of matched elements, optionally filtered by a selector.
