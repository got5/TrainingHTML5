• Ajax allows to dynamically make http requests with javascript. The request can then be used to add data to the DOM, to retrieve a JSON file and so on
  • jQuery provides several methods to do this, here are some of them.
    • jQuery.ajax({}): Is the general method, its behavior depends on options passed as parameter
    • jQuery.get({}): Is a shortcut of the option type:'GET' in the ajax() method, and so allows to use the GET method during the http request
    • jQuery.post({}): Is a shortcut of the option type:'POST' in the ajax() method, and so allows to use the POST method during the http request
    • jQuery.getJSON({}): Is a shortcut of the option dataType: 'json' in the ajax() method, and allows to reetrieve JSON content
  • jQuery allows you to specify the data you're expecting back: XML, HTML, JSON, JSONP, TEXT, SCRIPT
