• Most jQuery functions/plugins return the jQuery itself, basic mechanism is
    • $(selector/element/tag) returns the jquery object enhanced with matching elements
    • So the object returned contains both jQuery methods and results from the previous function
    • $(selector) -> {0:elt0, 1:elt1, func1:function(){}, ... }
    • So, we can do $(selector).func1() and if func1 return also jQuery $(selector).func1().func2()
