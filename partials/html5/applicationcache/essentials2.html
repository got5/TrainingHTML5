• A manifest specify cache rules through 3 parts
CACHE MANIFEST CACHE: /styles/main.css NETWORK: * FALLBACK: / /offline.html
  • cache list all resources that will be cached in the browser
  • network specifies which resources need a connection (here, everything but the cache ones)
  • fallback replace an unavailable resource by another one
  • A possible problem :
    • Some Browsers for mobile don't cache javascript library because it exceeds the maximum size defined
