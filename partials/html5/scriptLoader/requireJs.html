Hors sujet, modernizr doesn't expose by default its code as an AMD module (Asynchronous Module Definition ) . This can be done manually, but that's not sugegsted by modernizr. So use load instead(). RequireJS is an other Asset loader. It provide also a bit of dependency management.

  • Load require as any other js file
  • Just add the data-main attribute to tell require which script will contain the require configuration
<!doctype html> <html lang="en"> <head></head> <body> <!--....--> <script data-main="main" src="require.js"></script> </body> </html>
