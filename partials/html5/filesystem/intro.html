What is HTML5 File System ?

  • You can make CRUD operation on files and folders
  • You can read content of file
  • You can replace part or whole content of file

No built in resource inspector (like for localStorage or IndexedDB) for File System

  • Tool : Peephole (Chrome extension for file system api)
  • Other name : HTML5 FileSystem Explorer

Why use FileSystem API when indexedDB is able to store blobs as well?

  • Actually faster than indexedDB for large amounts of data
  • blobs storage is not yet implemented by chrome (work in progress)

This API is still experimental and some features may change in the future
