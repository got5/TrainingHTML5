• Specify the speed at which the animation progresses at different points within the animation
  • The default value is swing
  • Only Two implementations in jQuery : swing and linear
  • More easing functions are available with the use of plugins
  • Can specify an easing function per property
jQuery(document).ready(function(){ jQuery("#anime").off("click"); jQuery("#anime").on("click", function(event){ event.preventDefault(); jQuery("pre#editor").animate({ "height": 500 }, 500, "linear", function(){alert("the animation is finished!")}); }); });
