• HTML5 comes with new semantic elements.
  • Semantic means that these elements have a meaning, both for the developer and the browser
  • div and span are not semantic elements
  • while form, table and img are semantic elements, because they clearly define their contents
  • Notion of Semantic is important, because the document gives more information
    • Allows agents to understand the page, make correlation between pages
      • Example: the nav tag could help a search engine to display links directly in the result
    • Also important for the accessibility (audio browser)
