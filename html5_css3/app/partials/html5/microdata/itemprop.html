• itemprop allows to specify properties to a given itemtype
  • Detailled properties can be found by just following the url: http://schema.org/Movie
  • The previous example becomes:
<div itemscope itemtype ="http://schema.org/Movie"> <h1 itemprop="name">The Big Lebowski</h1> <span>Director: <span itemprop="director">Ethan Coen</span> </span> <span itemprop="genre">comedy</span> <a href="./url/to/trailer" itemprop="trailer">Trailer</a> </div>
