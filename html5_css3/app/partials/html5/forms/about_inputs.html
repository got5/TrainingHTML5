• New functionalities have appeared concerning input elements in the HTML5 forms
  • Note: supported by most browsers, and when not supported, default behavior is used (no error)
    • the placeholder attribute: a value to display in an input before the user starts typing
    • the autofocus attribute: place the focus on the concerned input when the page is loaded
