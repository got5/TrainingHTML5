• Dans le slide précédent, l'import est fait dans l'élément head
  • Les specs HTML4 (reste vrai en HTML5) disent ceci:
    The SCRIPT element places a script within a document. This element may appear any number of times in the HEAD or BODY of an HTML document.
  • Finalement, les spécifications HTML ne statuent pas, mais placer les scripts avant la fermeture du body = bonne pratique ( voir cet article )
<html> <head> ... </head> <body> ... <script src="js/script1.js" type="text/javascript" ></script> <script src="js/script2.js" type="text/javascript" ></script> </body> </html>
