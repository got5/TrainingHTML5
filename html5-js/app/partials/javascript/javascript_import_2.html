• Dans le slide précédent, l'import est fait dans l'élément head, et cela fonctionne bien.
  • Mais les specs HTML4 (reste vrai en HTML5) disent ceci:
    The SCRIPT element places a script within a document. This element may appear any number of times in the HEAD or BODY of an HTML document.
  • Finalement, les spécifications HTML ne statuent pas sur ce point, mais c'est une bonne pratique de placer les scripts en fin de balise body ( Cet article explique pourquoi )
<html> <head> ... </head> <body> ... <script src="js/script1.js" type="text/javascript" ></script> <script src="js/script2.js" type="text/javascript" ></script> </body> </html>
