• Multiple animation on a single element are not all run at the same time
  • jQuery add animation into a FIFO queue
  • Once an animation ends, it will trigger the next animation in the queue
  • jQuery use a series of Timeout() calls
  • Note: this is not true for show() and hide() unles you configure them
Animated div
jQuery(document).ready(function(){ jQuery("#anime").off("click").on("click", function(event){ jQuery(".animated") .show({duration: 0, queue: true}) .css("background", "red").animate({"height": 300}) .fadeOut().fadeIn() .hide({duration: 0, queue: true}); }); });
