Some methods also help to manipulate element styles.

  • jQuery("#div").addClass("cssClass1 cssClass2"): Adds the specified class(es) to each of the set of matched elements
  • jQuery("#div").removeClass("cssClass"): Remove a single class, multiple classes, or all classes from each element in the set of matched elements
  • jQuery("#div").toggleClass("cssClass"): Add or remove one or more classes from each element in the set of matched elements, depending on either the classâ€™s presence or the value of the switch argument
  • jQuery("#div").hasClass("cssClass"): Determine whether any of the matched elements are assigned the given class
  • jQuery("#div").css("border", "2px solid black"): Get the value of a style property for the first element in the set of matched elements or set one or more CSS properties for every matched element (an array can also be given as parameter)
