As their name says, attributes selectors allow to retrieve elements based on their attributes. Some matching patterns are available to get more flexibility.

  • a[rel] : contains the attribute rel
  • a[rel="friend"] : The attribute rel has the value friends
  • a[rel$="friend"] : The value of rel ends by friends
  • a[rel!="friend"] : If the attribute rel exists, its value must be different from friends
  • a[rel^="friend"] : The value of rel starts by friends
  • a[rel!="friend"][href^="http"] : condition1 and condition2
