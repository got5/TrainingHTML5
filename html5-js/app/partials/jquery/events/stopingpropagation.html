• A Handler may decide that event is fully processed and stop the bubbling
  • Two different ways :
    • W3C-compliant browers : event.stopPropagation()
    • And the second? return false;
  • Can disable the default behavior : event.preventDefault()
    Un exemple
<div class="d1">1 <div class="d2">2 <div class="d3">3 </div> </div> </div> jQuery(document).ready(function(){ jQuery(".d1").on("click", function(event){ event.stopPropagation(); alert("target = "+event.target.className+", this="+this.className); }); jQuery(".d2").on("click", function(event){ event.stopPropagation(); alert("target = "+event.target.className+", this="+this.className); }); jQuery(".d3").on("click", function(event){ event.stopPropagation(); alert("target = "+event.target.className+", this="+this.className); }); }); .d1 { background-color: green; } .d2 { background-color: blue; } .d3 { background-color: red; }
