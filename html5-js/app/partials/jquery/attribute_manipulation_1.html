Some methods can be used both to get data and set data, depending on their parameters. Here are some of them

  • jQuery("#id").attr("id") : Get the value of an attribute.
  • jQuery("#id").attr("id", "newVal") : Set the value of an attribute.
  • jQuery("#id").html() : Get the HTML content of the element
  • jQuery("#id").html("<p>New Block"</p>) : Set the HTML content of the element
  • jQuery("#id").val() : Get the value of a form input
  • jQuery("#id").val("New Value") : Set the value of a form input
  • jQuery("#id").width() : Get the size of the element
  • jQuery("#id").width(40) : Set the size of the element
