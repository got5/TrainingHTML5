• Firstly, we have to give a name to our plugin
  • Choose a name specific enough to be sure it will not be replaced by another one (don't name it myPlugin...)
  • Your plugin is just a function and will be executed when the function is called: you can define it as soon as jQuery is included, you don't need to wait the 'ready' event
  • A good practice is to define it in an auto calling function
( function($){ jQuery.fn.sayHello = function(){ console.log('Hey hello'); } } )(jQuery); $('#sayHello') .off('click').on('click', function(){ $('body').sayHello(); });
