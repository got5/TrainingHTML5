• If the second this is a standard element, you need to 'jQuerify' it to have access to the jQuery methods on it: $(this)
  • By doing return this.each(...), you also return the current jQuery instance which can then be chained with others plugins
( function($){ $.fn.sayHello = function(){ return this.each(function(index, element){ $(this).hide() .html("Hello jQuery") .css({"font-size": "80px"}) .slideDown(); }); } } )(jQuery); jQuery('#sayHello').off('click').on('click', function(){ console.log(jQuery('h1').sayHello().prop('tagName')); });
