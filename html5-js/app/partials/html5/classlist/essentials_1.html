1. classList : attribute (DOMTokenList) of an Element object
  2. Provide developers methods by which to add, remove and toggle CSS nodes
  3. Allows to check if a CSS class has been asigned to a given node
  4. For older browser, use these polyfills : classList.js and classList shim
  interface DOMTokenList (DOMTokenList) {
      readonly attribute unsigned long length;

      getter DOMString item(unsigned long index);
      boolean contains(DOMString token);
      void add(DOMString... tokens);
      void remove(DOMString... tokens);
      boolean toggle(DOMString token, optional boolean force);
      stringifier;
  }
