• To activate it, you have to use the manifest attribute on the html tag to point to the manifest url
  • The server has to answer to the request with the mime type: "text/cache-manifest"
  • No wildcard/regexp are allowed in the cache section, so how to say "I want to cache all images"?
    • Just generate the manifest dynamically on the server side
  • Use the manifest attribute makes this html page to be automatically cached
<!doctype html> <html lang="en" manifest="/offline.appcache"> ... CACHE MANIFEST #Version 00 CACHE: /js/jquery.min.js /js/app.min.js /styles/main.css /images/logo.png NETWORK *
