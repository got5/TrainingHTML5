• To activate it, you have to add the manifest attribute on the html tag and the relative url to the file as value
  • You have to set the correct mime type for the manifest file
    • In the request header, you can add :
      • "Content-Type":"text/cache-manifest" it depends on your server
  • No wildcard are allowed in the cache section !
<!doctype html> <html lang="en" manifest="/offline.appcache"> <head> <meta charset="UTF-8"> <title>Document</title> </head> <body> </body> </html> CACHE MANIFEST #Version 00 CACHE: /js/jquery.min.js /js/app.min.js /styles/main.css /images/logo.png NETWORK *
