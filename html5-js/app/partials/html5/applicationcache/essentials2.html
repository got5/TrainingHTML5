A manifest that tells the browser what files to store locally

CACHE MANIFEST CACHE: /styles/main.css NETWORK: *
  • The manifest is split in two parts:
    • The cache
      • list all assets that will be cached in the browser
    • The Network
      • list all assets that will be retrieved from the server

A possible problem :

  • Some Browsers for mobile don't cache javascript library because it exceeds the maximum size defined
