Now, if you have to make many feature tests in your application

  • There is a library to help you : Modernizr
  • What does it do ?
    1. Add classes indicating feature support, allowing you to apply style to elements
    2. Allow you to run feature detection
    3. Inject html5shiv
  • Install
    • Bower user ? bower install modernizr --save
    • Grunt user ? use grunt-modernizr
    • Want to check only some functionalities? use the custom builder
