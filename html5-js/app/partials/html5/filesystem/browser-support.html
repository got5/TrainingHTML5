• A the time begin (Summer 2014) only Webkit (Chrome, Opera and their mobile version) and BlackBerry implement this API
  • There are some polyfills and fallback out there we can use for unsupported Browser
    • Take a look at the idb.filesystem.js polyfill by Eric Bidelman (use Indexeddb)
  • Things to keep in mind
    • It 's not transactional. You can't be sure that the files are beeing saved successfully even if the callback success triggered
    • You can't search for files and in directory and file content in file
    • As all storage API, it's browser specific
