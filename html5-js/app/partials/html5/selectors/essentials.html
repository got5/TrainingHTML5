• Give the ability to find DOM elements using CSS selectors
  • Unify the complicated process of traversing/processing elements under a simple unified interface.
  • Similar to the jQuery Selector API
  • Provide only 2 methods : querySelector and querySelectorAll
  • The Efficiency of your selectors depends of the CSS3 support by the browser
  • Return a Static NodeList collection
partial interface Document { Element? querySelector(DOMString selectors); NodeList querySelectorAll(DOMString selectors); } partial interface DocumentFragment { Element? querySelector(DOMString selectors); NodeList querySelectorAll(DOMString selectors); } partial interface Element { Element? querySelector(DOMString selectors); NodeList querySelectorAll(DOMString selectors); }
