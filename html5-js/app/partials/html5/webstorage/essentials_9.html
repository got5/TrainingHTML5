1. Security
    • Web Storage objects are regulated by the same origin policy
    • http://example.com/ and http://example.com/about will share storage
    • http://example.com/ and https://example.com:87/do not share storage
  2. Quota (around 5Mb, varies from device to device, browser to browser)
    • Exceed quote : throw QUOTA_EXCEEDED_ERR and SECURITY_ERR
    • code : 22
  3. Potential Use Cases
    • Storing user preferences
    • Caching image data for photo galleries or maps, ...
