• Drag from an element to another one
  • Set the draggable attribute to true and use the dragstart event on the source element
  • Use the dragover and drop events on the target element respectively to allow the drop action and to implement this action behavior
  • Finally, use the dataTransfer attribute to transfer programmatically data between the source and the target
  • caniuse.com
<div id="div1" style="border: solid 1px black" ondrop="drop(event)" ondragover="allowDrop(event)">target</div> <div id="drag1" draggable="true" style="border: solid 1px black" ondragstart="drag(event)">source</div> allowDrop= function(ev) { ev.preventDefault(); } drag= function(ev) { ev.dataTransfer.setData("Text", ev.target.id); } drop= function(ev) { ev.preventDefault(); var data = ev.dataTransfer.getData("Text"); console.log(data); } console.log('loaded');
