• Using Range to add constraint to the result :
    • only(String) : only the result that respect this condition (on String)
    • next, prev, nextunique, prevunique : order and restrict result
//... var name = "Johnson", transaction = db.instance.transaction(['people'],'readonly'), index = transaction.objectStore('people').index('lastName'), range = IDBKeyRange.only(name); //range = IDBKeyRange.lowerBound(name,true); //works with string too var getRequest = index.openCursor(range,'prev'); //reverse the order of the result //var getRequest = index.openCursor(range,'next'); //default var getRequest = index.openCursor(range,'prevunique'); //same as above but restrict to a single result var getRequest = index.openCursor(range,'nextunique'); //...
