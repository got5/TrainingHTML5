• Using Range to add constraint to the result :
    • upperBound(NUMBER/STRING,Boolean) : the boolean is for exclude
    • lowerBound(18,true)
    • bound(NUMBER/STRING,NUMBER/STRING,Boolean,Boolean)
      • ex : bound(18,26,true,false): 18 < result <= 26
