indexedDB is a Data Store

  • Contains Objects Stores
  • Similar to noSql Databases document oriented (like a simplified version of MongoDB)
    • Store an object as is (object store...)
    • Use indexes (allows to retrieve data based on a value of the index)
    • Store data as pair of (key -> value) (like the localStorage, but the value is an object)
    • enforce security origin
    • Database is named and specific to an application
    • Has a version number (integer or string)
