Modernizr includes (optionally) a script loader (Yepnope.js).

You call Yepnope with yepnope() or Modernizr.load()

The scripts are loaded asynchronously and in parallels !

Exemple : test if the browser support geolocation and load the proper polyfill if it doesn't.

Modernizr.load({ test: Modernizr.gelocation, yep: 'geo.js', nope: 'geo-polyfill.js', });

Follow the Guide in the yepnope documentation (callback and complete functions)
