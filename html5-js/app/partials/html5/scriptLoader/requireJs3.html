• The define works like require()
    • can have an extra param (first position). The name of the module.
  • We use the define function to create a new module.
  • It has to return something, which will be injected to the callback function
  • Usually, we will be returning a function that exposes variables and/or methods
define(["jquery"], function($){ var module = function(){ this.oneFunction = function(){ //... } //... }; return module; });
