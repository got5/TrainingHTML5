• The define function allows to create a new module
  • A module is an object which exposes variables/functions and which can depend on other modules
  • can have an extra param (first position). The name of the module.
  • In the below example, we are defining a new module named /path/to/module and which depends on the jquery module
define(["jquery"], function($){ return { myProp: 'propValue', myFunction: function(){ //... } } });
